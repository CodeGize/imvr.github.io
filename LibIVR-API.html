Install Instructions

install automatically

  • Download sdk.zip package.
  • Uncompress sdk.zip.
  • Run "IVR_install.bat" as administrator.

install manually

  • Download dll files which architecture(x86 or x64) you need.
  • Mark sure "libIVR.dll" file in your "%SystemRoot%\system32" folder.(NOTE: architecture x64 need in system32, x86 need in syswow64).

Basic Module

Method name Comment Parameter Return
int IVR_Init() 初始化 1 - repeat init;
0 - successed;
-1 failed;
int IVR_Deinit() 卸载 1 - repeat deinit;
0 - successed;
-1 failed;
int IVR_SetDisplayCapturerPath(const char *path) 设置双屏程序文件路径 path: 双屏程序路径，注意 \ 需要转义
int IVR_StartDisplayCapturer() 启动双屏功能 -2 The "ScreenCapturer.exe" file can't be found.
-1 - failed;
0 - successed;
int IVR_StopDisplayCapturer() 关闭双屏功能 -1 - failed
0 - successed

Eyeshield

Method name Comment Parameter Return
int IVR_EnableEyeshield(int level) 启动护眼功能 level: 1 - 一级护眼 （目前只有一级，该值传入1即可）； -3 - 未初始化;
-2 - 映墨头显设备未发现；
-1 - 失败；
0 - successed；
int IVR_DisableEyeshield() 关闭护眼功能 同上
